  2. 冥想打坐，寻找丹田
  3. 让丹田的气越来越大
  4. 看内经图

- OKR 2 滑板
  1. 练习站立power slid，有点小进步但是不多
  2. 练习双手扶地power slid，成功
  3. 练习toside单手扶地power slid，完成百分之九十
  4. 练习滑板动作去的热身要领，要让身体热起来，自己情绪热起来
  5. 发现手套slid的几个练习要点，1.场地（需要一个比较陡的坡，不用滑很远，稍微有点速度就可以，平底或者坡度不够，练习起来有点难）2.先练习双手，再来单手。
